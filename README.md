# Reestar – Simple Food Website

Reestar is a basic responsive food ordering website built using HTML and CSS. This project displays a collection of food items with an order option and a contact form at the bottom. It's ideal for beginners learning frontend development.

---

## Features

- Responsive layout using HTML & CSS  
- Four sample food items with images  
- "Order Now" buttons under each item  
- A "Contact Us" form with Name, Age, and Email fields  
- Simple navigation links (Home, Recipe, Contact)

---

## Files Included

- index.html – Main HTML structure of the website  
- style.css – Styling file for layout, fonts, and responsiveness  
- screenshot.png – (Optional) Preview image of the website
